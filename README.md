# ai-drone-face-recognition
Developed an intelligent UAV system integrating AI-based facial recognition with robotic actuation for targeted payload delivery. The system combines Raspberry Pi for real-time image processing, OpenCV for face detection, Arduino for robotic arm control, and Pixhawk for autonomous flight, enabling efficient operations in critical scenarios.

## Tech Stack
- Python
- OpenCV
- Raspberry Pi 4B
- Arduino (Portenta H7)
- Pixhawk Flight Controller
- PX4 Autopilot
